USB / Disk Repair

Rebuild-USB.ps1

Completely wipes and rebuilds a USB drive (or other disk) using MBR and formats it as NTFS/FAT32. Use when a drive shows 0 bytes, RAW, or corrupted partitions.

Path: G:\Scripts\DiskTools\Rebuild-USB.ps1

Run in PowerShell:
G:\Scripts\DiskTools\Rebuild-USB.ps1 -DiskNumber 3 -FileSystem NTFS
